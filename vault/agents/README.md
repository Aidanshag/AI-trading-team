# Agent standing briefs

One note per agent, where long-lived observations and preferences accumulate. These are *separate* from the system prompts under `/agents/*.md`:

- **`/agents/*.md`** (in the project root) = the system prompt. Loaded every wake. Should be stable; edit when you want to change an agent's fundamental behavior.
- **`vault/agents/{agent_name}.md`** = the agent's evolving journal. Loaded on wake as additional context. Accumulates lessons, idiosyncratic quirks, flagged user preferences. The agent appends; the user curates.

Think of the first as the employment contract and the second as the work notebook.
