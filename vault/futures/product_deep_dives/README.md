# Product deep-dives

One note per futures product. This is the desk's long-lived product knowledge — specs, drivers, typical patterns, edge cases. Analysts consult these on every wake; PM and Risk reference them for sizing and sanity-checks.

## Rules

- **One note per symbol**, named `{SYMBOL}.md` (e.g. `CL.md`, `ES.md`, `6E.md`).
- **Kept current** by the sector analyst that covers it.
- **Reviewed quarterly** — calendar quirks, expiration rules, and microstructure evolve.
- **Template**: `TEMPLATE.md` in this folder. Use it for new deep-dives.
