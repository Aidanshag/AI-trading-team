# Equities weekly reviews

Every Sunday evening, each equity analyst reviews their week's shadow trades against actual price action. Equity PM aggregates; Risk Manager scores process quality.

## File naming

`weekly_{YYYY-MM-DD}.md` — one file per week, dated to the Friday close.

## Template

```markdown
---
type: review
date: YYYY-MM-DD
author: Equity PM
---

# Equities weekly review — week of {date}

## Scorecard
- Shadow trades this week: N
- Win rate: X%
- Avg R: Y
- By analyst: ...

## What worked
- ...

## What didn't
- ...

## Calibration notes
- Were our stops realistic? Sizing correct for conviction? Pattern selection matching regime?

## Refinement asks to user
- ...
```
